# Task Management System

An ASP.NET Core MVC application demonstrating full CRUD (Create, Read, Update, Delete) functionality using an in-memory data store. This project focuses on clean MVC architecture, Razor views, and model validation.

## Features
- Create, view, edit, and delete tasks
- Task fields include title, description, status, and due date
- Server-side validation using Data Annotations
- User-friendly forms with validation feedback
- Confirmation prompt before deleting a task

## Technologies Used
- ASP.NET Core MVC (.NET 8)
- C#
- Razor Views
- Model Validation (Data Annotations)
- Visual Studio
- Git & GitHub

## Project Purpose
This project was built to strengthen understanding of ASP.NET Core MVC fundamentals and CRUD operations before integrating a database layer. It demonstrates proper separation of concerns and clean controller logic.

## Future Enhancements
