# Task Management System

A simple ASP.NET Core MVC application demonstrating full CRUD (Create, Read, Update, Delete) functionality using an in-memory data store. This project focuses on MVC architecture, controller logic, and Razor views without database integration.

## Features
- Create new tasks
- View a list of tasks
- Edit existing tasks
- Delete tasks
- Task properties include title, description, status, and due date

## Technologies Used
- ASP.NET Core MVC (.NET 8)
- C#
- Razor Views
- Visual Studio
- Git & GitHub

## Purpose
This project was built to strengthen understanding of ASP.NET Core MVC fundamentals and CRUD operations before integrating a database layer. It serves as a foundation for future enhancements such as SQL Server or MySQL integration.
